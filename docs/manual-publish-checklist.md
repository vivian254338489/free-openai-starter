1. Push this local repo to the target GitHub repository
2. Set repository title to match the README title
3. Set repository description from `docs/github-launch-pack.md`
4. Add topics from `docs/github-seo.md`
5. Pin the best architecture or routing screenshot later

2. Record one face-to-camera explanation if possible
3. Use scripts from `docs/tiktok-launch-pack.md`
4. Put repo link or landing path in bio
