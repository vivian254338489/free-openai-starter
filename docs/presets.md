# Presets

This repository includes small starter presets for common OpenAI-compatible setups.

## TKEN Kimi

File:

```text
config/presets/tken-kimi-free.yaml
```

Use it when you want the fastest path to a free first request with `kimi`.

## TKEN Minimax

File:

```text
config/presets/tken-minimax-free.yaml
```

Use it when you want a preset focused on `minimax`.

## Why Presets Matter

Presets reduce setup friction for new users:

- fewer manual edits
- clearer provider-specific examples
- faster time to first successful request
